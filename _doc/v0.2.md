# py4vm v0.2 設計文件

## 概述

v0.2 版本在 v0.1 的基礎上加入了 **模組 import 功能**，可以載入標準函式庫模組並使用其匯出的符號。

## 新增功能

### 模組系統

```
+------------------+
|   模組資料夾     |
| (modules HashMap) |
+------------------+
        |
        v
+-------------+  +-------------+  +-------------+
|    sys      |  |    math     |  |     os      |
| version    |  | pi         |  | name       |
| maxsize    |  | e          |  | sep        |
+-------------+  +-------------+  +-------------+
```

### 預設模組

| 模組 | 匯出符號 | 說明 |
|------|----------|------|
| sys | version | 版本字串 "0.2.0" |
| sys | maxsize | i64::MAX |
| math | pi | 3.14159265359 |
| math | e | 2.71828182846 |
| math | tau | 6.28318530718 |
| os | name | "posix" |
| os | sep | "/" |
| os | linesep | "\n" |

## 新增指令

### Import

```rust
Instruction::Import(String)
// 語法: import <module_name>
// 將模組的所有匯出符號加入 locals
```

執行流程：
1. 在 modules HashMap 中查詢模組
2. 將模組的 exports 複製到 locals

### FromImport

```rust
Instruction::FromImport(String, String)
// 語法: from <module_name> import <symbol_name>
// 將指定符號壓入堆疊
```

執行流程：
1. 在 modules HashMap 中查詢模組
2. 取得指定符號的值
3. 將值壓入堆疊

## VM 結構變更

```rust
struct Module {
    name: String,
    code: Vec<Instruction>,
    exports: HashMap<String, Op>,  // 模組匯出的符號
}

struct VM {
    stack: Vec<Op>,
    locals: HashMap<String, Op>,
    modules: HashMap<String, Module>,  // 新增：模組資料表
    code: Vec<Instruction>,
    pc: usize,
}
```

## 測試範例

### import

```python
import sys
import math
import os
print(sys.version)   # 0.2.0
print(math.pi)      # 3.14159265359
print(os.name)     # posix
```

對應指令：
```
Import("sys")
Import("math")
Import("os")
LoadName("version"), Call("print", 1)
LoadName("pi"), Call("print", 1)
LoadName("name"), Call("print", 1)
Return
```

輸出：
```
0.2.0
3.14159265359
posix
```

### from_import

```python
from math import pi
from math import e
print(pi, e)
```

對應指令：
```
FromImport("math", "pi")
FromImport("math", "e")
Call("print", 2)
Return
```

輸出：
```
3.14159265359 2.71828182846
```

### mix

```python
import math
print(math.pi * 2)
```

對應指令：
```
Import("math")
LoadName("pi")
LoadInt(2)
BinaryMultiply
Call("print", 1)
Return
```

輸出：
```
6.28318530718
```

## 使用方式

```bash
# 執行測試
bash test.sh

# 個別測試
./target/debug/py4vm import
./target/debug py4vm from_import
./target/debug py4vm mix
```

## 限制

1. 模組需預先定義，無法動態載入
2. 沒有 `as` 別名
3. 沒有 `*` 萬用匯入
4. 沒有相對/絕對 import 路徑
5. 模組沒有程式碼，只有資料
