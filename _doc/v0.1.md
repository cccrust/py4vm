# py4vm v0.1 設計文件

## 概述

py4vm 是一個用 Rust 實現的簡易 Python 虛擬機器，能夠執行自訂的 Bytecode 指令。這個版本實現了基本的堆疊機器功能，包括運算、比較、流程控制等核心特性。

## 系統架構

```
+------------------+     +------------------+     +------------------+
|   原始碼/指令     | --> |   py4vm 直譯器   | --> |   執行結果      |
|   (Vec<Inst>)   |     |   (VM 結構)     |     |   (Op)         |
+------------------+     +------------------+     +------------------+
```

## 核心類型

### Op 值類型

```rust
#[derive(Clone)]
enum Op {
    None,           // Python None
    Bool(bool),     // 布林值
    Int(i64),      // 64位元整數
    Float(f64),    // 64位元浮點數
    String(String), // 字串
    List(Vec<Op>), // 列表
}
```

### 指令類型

```rust
#[derive(Clone)]
enum Instruction {
    LoadConst(Op),      // 載入常數
    LoadInt(i64),       // 載入整數常數
    LoadFloat(f64),     // 載入浮點常數
    LoadString(String), // 載入字串常數
    StoreName(String),  // 儲存到變數
    LoadName(String),  // 載入變數
    BinaryAdd,         // 加法 (+)
    BinarySubtract,    // 減法 (-)
    BinaryMultiply,     // 乘法 (*)
    BinaryDivide,       // 除法 (/)
    BinaryModulo,      // 取餘 (%)
    CompareOp(i32),     // 比較運算
    JumpIfFalse(usize), // 條件跳躍 (false)
    JumpIfTrue(usize),  // 條件跳躍 (true)
    Jump(usize),       // 無條件跳躍
    Call(String, usize),// 函式呼叫
    Return,            // 返回
}
```

## 虛擬機器設計

### VM 結構

```rust
struct VM {
    stack: Vec<Op>,                // 運算元堆疊
    locals: HashMap<String, Op>,      // 區域變數
    code: Vec<Instruction>,       // 指令序列
    pc: usize,                  // 程式計數器
}
```

### 執行流程

1. 取出 `code[pc]` 指令
2. `pc += 1`
3. 執行指令
4. 重複直到遇見 `Return` 或 `pc >= code.len()`

### 堆疊操作

```
執行前堆疊:     [a, b]
BinaryAdd  -->  [a+b]
執行後堆疊:     [a+b]
```

## 指令說明

### 資料載入

| 指令 | 說明 | 範例 |
|------|------|------|
| LoadInt(n) | 將整數 n 壓入堆疊 | `LoadInt(5)` → stack: [5] |
| LoadString(s) | 將字串 s 壓入堆疊 | `LoadString("hi")` → stack: ["hi"] |
| LoadName(n) | 將變數 n 的值壓入堆疊 | a=5 → LoadName("a") → stack: [5] |
| StoreName(n) | 彈出堆疊頂端存入變數 | stack: [5] → StoreName("x") → x=5 |

### 二元運算

| 指令 | 說明 | 語法對應 |
|------|------|----------|
| BinaryAdd | 彈出兩值相加後推入 | a + b |
| BinarySubtract | 彈出兩值相減後推入 | a - b |
| BinaryMultiply | 彈出兩值相乘後推入 | a * b |
| BinaryDivide | 彈出兩值相除後推入 | a / b |
| BinaryModulo | 彈出兩值取餘數後推入 | a % b |

### 比較運算

| 參數 | 說明 | 範例 |
|------|------|------|
| 0 | 小於 (<) | 3 < 5 → true |
| 1 | 小於等於 (<=) | 3 <= 3 → true |
| 2 | 等於 (==) | 3 == 3 → true |
| 3 | 不等於 (!=) | 3 != 3 → false |
| 4 | 大於 (>) | 3 > 5 → false |
| 5 | 大於等於 (>=) | 3 >= 3 → true |

### 流程控制

| 指令 | 說明 |
|------|------|
| Jump(addr) | pc = addr，無條件跳躍 |
| JumpIfFalse(addr) | 若堆疊頂為 false，pc = addr |
| JumpIfTrue(addr) | 若堆疊頂為 true，pc = addr |
| Return | 返回堆疊頂端值 |

### 函式呼叫

```rust
Call("print", n)
// 彈出 n 個參數，倒轉順序後呼叫內建函式
```

支援的內建函式：
- `print`: 輸出��換行
- `len`: 回傳長度

## 測試範例

### hello

```
LoadString("Hello, World!") → load
Call("print", 1)           → print
Return
```

輸出：`Hello, World!`

### add

```
LoadInt(1)        → 1
LoadInt(2)        → 1, 2
BinaryAdd        → 3
Call("print", 1) → print
Return
```

輸出：`3`

### loop

```
LoadInt(0)                    → i=0
StoreName("i")                
Jump(2)                       → 跳到迴圈開始
LoadName("i")                 → load i
Call("print", 1)             → print i
LoadName("i")                 → load i
LoadInt(1)                    → i, 1
BinaryAdd                    → i+1
StoreName("i")                → i=i+1
LoadName("i")                 → load i
LoadInt(5)                    → i, 5
CompareOp(0)                  → i<5
JumpIfFalse(13)               → 跳出
Jump(2)                       → 迴圈
Return
```

輸出：`0 1 2 3 4`

### ifelse

```
LoadInt(5)
StoreName("x")
LoadName("x")
LoadInt(3)
CompareOp(4)           // x > 3
JumpIfFalse(9)         // 跳到 else
LoadString("x > 3")
Call("print", 1)
Jump(11)               // 跳到結束
LoadString("x <= 3")
Call("print", 1)
Return
```

輸出：`x > 3`

### arith

```
LoadInt(10), LoadInt(3), BinaryAdd, Call(print)  → 13
LoadInt(10), LoadInt(3), BinarySubtract       → 7
LoadInt(10), LoadInt(3), BinaryMultiply       → 30
LoadInt(10), LoadInt(3), BinaryDivide       → 3.33
LoadInt(10), LoadInt(3), BinaryModulo      → 1
Return
```

輸出：`13 7 30 3.33... 1`

## 實作細節

### 整數溢位處理

使用 `saturating_add` 避免溢位：

```rust
Op::Int(i1.saturating_add(i2))
```

### 浮點除法

```rust
if i2 != 0 {
    Op::Float(i1 as f64 / i2 as f64)
} else {
    Op::None
}
```

## 使用方式

```bash

cargo build

# 執行測試
bash test.sh

# 直接執行某個測試
./target/debug/py4vm hello
./target/debug/py4vm add
./target/debug/py4vm loop
```

## 的限制

這個版本存在以下限制：

1. 不支援真正的 .pyc 檔案載入
2. 整數只能處理 i64 範圍
3. 沒有完整的標準函式庫
4. 沒有異常處理機制
5. 沒有物件導向特性
